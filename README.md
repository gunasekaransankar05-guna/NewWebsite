# Anniversary Website Customization Guide

## 🎵 Music
1.  Place your music file in `assets/music/`.
2.  Rename it to `bg-music.mp3` OR update `index.html` line 22 to match your filename.

## 📹 Videos
1.  Place your video files in `assets/videos/`.
2.  Rename your main memory video to `memory-clip.mp4` OR update `index.html` line 77.

## 📸 Images
-   Replace the Unsplash URLs in `index.html` with your own local paths (e.g., `assets/images/photo1.jpg`).
-   To add more photos, duplicate the `<div class="gallery-item">...</div>` block in `index.html`.

## 💌 Letters
-   Edit the text inside `<div class="text">...</div>` in `index.html` (lines 88-92) to write your own message.
